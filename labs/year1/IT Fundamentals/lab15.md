Lab description: Students will install the pfSense virtual firewall and setup a private network segment for the 3 previously created VMs.  This will be the firewall students will work with for the remainder of the year.  Student will create a custom subnet for their LAN network segment.

Standards: CompTIA IT Fundamentals 2.7 Explain basic networking concepts. <br>
           CompTIA Security+ 2.1 Install and configure network components, both hardware-
              and software-based, to support organizational security. <br>
           CompTIA Security+ 3.2 Given a scenario, implement secure network architecture concepts.
